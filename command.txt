네트워크 관련 명령어

ifconfig : 네트워크 인터에이스 설정, 확인 ip주소, 넷마스크값, MAC주소 확인 가능
route : 라우팅 테이블의 정보 출력, 관리, 네트워크 주소, 게이트웨이 주소를 확인, 설정
netstat : 네트워크의 연결 상태 출력
arp : 캐시 관리, ARP 캐시는 현재 접속되어있는데 32비티 IP 주소를 하드웨어 주소로 바꿔서 기억하는 곳(로컬 MAC주소 확인)
ping : 인터넷 제어 메세지 프로토콜인 ICMP을 이용하여 네트워크 연결 확인
traceroute : 패킷이 특정 호스트까지 라우팅되는 과정을 출력함
nslookup : DNS를 이용하여 도메인이나 IP 조회
dig : 도메인명으로 정보를 조회
host : 도메인명으로 정보를 조회
hostname : 시스템에 설정된 호스트네임을 출력하거나 설정
mii-too : 네트워크 인터페이스의 상태를 점검하고 설정하는 유틸리티
ethtool ethx : 이더넷 카드 설정 정보 출력, 변경
ip : 이더넷 장치, IP주소, 라우팅 정보 등 설정 정보 출력, 변경 게이트웨이 주소값 설정 및 삭제
ss : 소켓 상태를 출력해주는 명령 (netstat랑 유사, 더 많은 정보 제공)

--사용자관리--
useradd : 사용자를 추가
useradd -D : /etc/default/useradd를 보여줌(계정 생성 시 기본적으로 적용되는 설정이 들어있는 파일)
useradd -g : 기본 그룹 지정
useradd -b : 사용자 홈 디렉터리의 상위 디렉터리를 지정
useradd -e : 계정 만료일 지정
useradd -s : 사용자의 기본 쉘 지정
passwd [사용자명] : 등록된 사용자의 암호를 지정, 변경
su [옵션] [사용자명] : 사용자를 전환할 때 사용
grep [문자열] : 문자열 검색
pwck : Password Check로 /etc/passwd와 /etc/shadow를 점검해 줌
useradd -D d: /etc/default/useradd를 보여줌(계정 생성 시 기본적으로 적용되는 설정이 들어있는 파일)
usermod : 사용자의 쉘, 홈 디렉터리, 그룹, UID 등을 변경, 사용자 관련 대부분 정보 변경 가능
usermod -u : 사용자의 UID값 변경
usermod -l : 사용자 아이디 변경
userdel : 사용자 계정 삭제
userdel -r : 사용하던 홈 디렉터리 안의 모든 것 다 삭제
passwd -l : 패스워드에 잠금을 설정해 로그인 막음
passwd -u : 패스워드 잠금 해제
passwd -d : 패스워드 제거(걍 로그인댐)
chage : 사용자 패스워드에 대한 정보를 출력하고 설정, /etc/shadow의 날짜 관련 필드 설정 모두 할 수 있음
groupadd : 새로운 그룹 생성, root만 사용 가능함
groupmod : 그룹명이나 GID 변경할 때 사용
groupmod -n : 그룸 이름 변경
groupmod -g : 그룹 GID 변경
groupdel : 그룹 삭제
gpasswd : 그룹 패스워드 설정, 그룹관리자 지정
gpasswd -A : root가 그룹 관리자를 지정할 때
gpasswd -a : 그룹 관리자가 그룹에 사용자 추가
gpasswd -d : 그룹 관리자가 그룹에서 사용자 제외
newgrp [그룹명] : 사용자가 일시적으로 1차 그룹을 변경할 때 사용하는 명령
users : 시스템에 로그인되어 있는 사용자의 아이디를 출력해 주는 명령이다.
who : 시스템에 로그인되어 있는 사용자를 출력해 주는 명령어
w : 시스템에 로그인되어 있는 사용자와 사용자가 수행중인 작업을 출력해 주는 명령어
logname : 사용자의 로그인 계정을 출력해 준다
id : 시스템에 등록된 아이디에 대한 정보를 출력, UID, GID, 속한 그룹 정보를 보여줌 
lslogins : 시스템 전체 사용자의 정보를 출력해 주는 명령어 모든 곳에서 관련정보 다 출력해줌 

--파일시스템관리--
ls -l : 권환 확인 가능
chmod : 파일이나 디렉터리에 접근할 수 있는 허가관을 설정하는 명령
chown : 파일이나 디렉터리의 소유권 및 그룹 소유권을 변경하는 명령
chgrp : 파일이나 디렉터리의 소유 그럽으로 바꾸는 명령(주로 root가 사용0
umask : 파일이나 디렉터리 생성 시 부여되는 기본 허가권 값을 지정하는 명령
ls -i : 파일, 디렉터리의 I-node 번호 확인 
ln : 하드링크생성, 옵션 붙이면 심볼릭 링크 생성
fdisk -l : 하드디스크의 종류 및 위치 확인(root 계정으로)
mkfs : 새롱누 파일 시스템 만드는 명령어, root만 사용 가능, 유형 지정 안하면 ext2로 생성됌
mke2fs : ext2, ext3, ext4 파일 시스템을 만드는 명령어(최신 사용), 유형 지정 안하면 ext2로 생성됌
mount : 보조기억장치나 파일 시스템이 다른 디스크를 /의 하위 디렉터리로 연결하여 사용 가능하게 해 주는 명령어
unmount : 마운트 해제시켜줌, 반드시 사용해야 CD 등 제거 가능(usb)
eject : CD-ROM, DVD등과 같은 미디어 꺼낼 때 사용, 자동으로 unmount 작업 수행
